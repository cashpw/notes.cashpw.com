Main diagonal

In linear algebra, the main diagonal (sometimes principal diagonal, primary diagonal, leading diagonal, major diagonal, or good diagonal) of a Matrix \(A\) is the list of entries \(A_{i,j}\) where \(i=j\). All off-diagonal elements are zero in a diagonal matrix. Surjective function

In mathematics, a surjective function (also known as surjection, or onto function) is a function f that maps an element x to every element y; that is, for every y, there is an x such that f(x) = y. In other words, every element of the function’s codomain is the image of at least one element of its domain. Big-Omega notation

Big Omega notation, written as \(\bigomega{g(n)}\), indicates an worst-case asymptotic lower bound. While \(2n^2\) is in \(\bigomega{1}\), it’s preferred to be as precise as possible and state the lowest valid case: \(2n^2 = \bigomega{n^2}\). Little-o notation

Little-o notation, written as \(\littleo{g(n)}\), is a stronger statement than Big O notation. It implies that \(g(x)\) grows much faster than \(f(x)\).
