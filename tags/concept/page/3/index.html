TODO: Validate indices heap_start, heap_end = heap_range parent_index = heap_start while True: child_a_index, child_b_index = get_child_indices(parent_index) swap_index = parent_index if child_a_index <= heap_end and comparator(items[child_a_index], items[parent_index]) > 0: swap_index = child_a_index if child_b_index <= heap_end and comparator(items[child_b_index], items[parent_index]) > 0 and comparator(items[child_b_index], items[child_a_index]) > 0: swap_index = child_b_index if swap_index == parent_index: break swap(swap_index, parent_index) parent_index = swap_index def swap(index_a: int, index_b: int) -> None: # Fixed mindset

Those with a fixed Mindset believe “intelligence is static” and there is very little to be done to improve ability. Feedback is seen as an “evaluation of their underlying ability” since success is seen only as a result of this innate ability, not the effort put in. Failure is much more intimidating since it “suggests constraints or limits they would not be able to overcome”. Meaningful rest

Meaningful rest is a type of rest which leaves the individual feeling rejuvenated. It exists in contrast to default actions – “the things that take no activation energy to start, the things [one] reflexively reaches for” (Nanda n.d.). The term is from an article by Neel Nanda.
