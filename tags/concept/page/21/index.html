Priority queue

In computer science, a priority queue is an abstract data-type similar to a regular queue [Queue] or stack data structure in which each element additionally has a “priority” associated with it. In a priority queue, an element with high priority is served before an element with low priority. Skewed binary tree

A skewed binary tree is a pathological/degenerate tree in which the tree is either dominated by the left nodes or the right nodes. Thus, there are two types of skewed binary tree: left-skewed binary tree and right-skewed binary tree. (Programiz n.d.)
