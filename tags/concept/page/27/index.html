Zero-sum game

Zero-sum game is a mathematical representation in Game theory and economic theory of a situation which involves two sides, where the result is an advantage for one side and an equivalent loss for the other. In other words, player one’s gain is equivalent to player two’s loss, therefore the net improvement in benefit of the game is zero. (“Zero-Sum Game” 2022) The opposite of a Non-zero-sum game.
