Anatomy of a URL

Back¶

Using http://www.a.com:8080/b/c?d=e&f=g#h as an example:

  1. Scheme (e.g. http)
  2. Authority (e.g. www.a.com:8080)
    1. Host (e.g. www.a.com)
    2. Port (e.g. 8080)
  3. Path (e.g. /b/c)
  4. Parameters (e.g. ?d=e&f=g)
  5. Anchor (e.g. #h)
