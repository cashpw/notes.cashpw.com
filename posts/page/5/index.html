Centering matrix

In mathematics and multivariate statistics, the centering matrix is a symmetric and Idempotent matrix, which when multiplied with a vector has the same effect as subtracting the mean of the components of the vector from every component of that vector. \(C_n = I_n - \tfrac{1}n J_n\) (“Centering Matrix” 2021) References .csl-entry{text-indent: -1.5em; margin-left: 1.5em;}“Centering Matrix.” 2021. Wikipedia, June. https://en.wikipedia.org/w/index.php?title=Centering_matrix&oldid=1027624781.

De-meaned vector

\(\tilde{a} \overset{\Delta}{=} \vec{a} - \operatorname{avg}(\vec{a})\vec{1} = \mathbf{C}\vec{a}\) (Boyd and Vandenberghe 2018, 60) Note that \(\mathbf{C}\) is the Centering matrix.
