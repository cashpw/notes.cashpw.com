> In linear algebra, an idempotent matrix is a [Matrix]({{< relref "matrix.md" >}}) which, when multiplied by itself, yields itself. That is, the matrix \\(A\\) is idempotent if and only if \\(A^2 = A\\).
>
> (“Idempotent Matrix” 2022)
