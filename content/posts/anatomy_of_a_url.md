Using `http://www.example.com:8080/foo/bar?baz=boo&color=blue#cats` as an example (MDN n.d.):

| Name       | From the example       |
|------------|------------------------|
| Scheme     | `http`                 |
| Authority  | `www.example.com:8080` |
| Host       | `www.example.com`      |
| Port       | `8080`                 |
| Path       | `/foo/bar`             |
| Parameters | `?baz=boo&color=blue`  |
| Anchor     | `#cats`                |
