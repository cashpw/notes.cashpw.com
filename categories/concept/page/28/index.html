Bellman-Ford algorithm

The Bellman–Ford algorithm is an algorithm that computes shortest paths from a single source vertex to all of the other vertices in a weighted digraph. It is slower than Dijkstra’s algorithm for the same problem, but more versatile, as it is capable of handling graphs in which some of the edge weights are negative numbers. Widest path problem

In graph algorithms, the widest path problem is the problem of finding a path between two designated vertices [Vertices] in a weighted graph [Graph], maximizing the weight of the minimum-weight edge [Edge] in the path. The widest path problem is also known as the maximum capacity path problem.
