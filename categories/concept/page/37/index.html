Widest path problem

In graph algorithms, the widest path problem is the problem of finding a path between two designated vertices [Vertices] in a weighted graph [Graph], maximizing the weight of the minimum-weight edge [Edge] in the path. The widest path problem is also known as the maximum capacity path problem.
